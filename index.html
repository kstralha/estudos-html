Introdução

(Voltar)

HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental para a criação e estruturação de páginas da web. O HTML permite que os navegadores interpretem e exibam informações como texto, imagens, links, vídeos e outros elementos multimídia de maneira coerente e organizada. Em resumo, o HTML é a base da web moderna, permitindo a estruturação de conteúdo e o fornecimento de informações em um formato que os navegadores podem interpretar e exibir de maneira coerente e organizada.

Tags

(Voltar)
  • Head:

    Contém metadados sobre o documento, como título, codificação e links para estilos e scripts.

  • Title:

    Define o título da página exibido na aba ou barra de título do navegador.

  • Link:

    Liga o documento a folhas de estilo externas.

  • Script:

    Utilizado para adicionar scripts, como JavaScript, para interatividade e funcionalidade.

  • Body:

    Contém o conteúdo visível da página, como texto, imagens e elementos interativos.

  • H1 a H6:

    Define títulos de diferentes níveis de importância (1 sendo o de maior nível hieráriquico e 6 sendo o menor).

  • P:

    Cria um parágrafo de texto.

  • A:

    Cria links para outras páginas ou recursos.

Essas são algumas delas.

Client-side

(Voltar)

O "client side" em HTML refere-se à parte do processamento que ocorre no navegador do usuário, ou seja, no lado do cliente. No contexto do desenvolvimento web, o client side envolve a manipulação e renderização de elementos visuais e interativos diretamente no navegador, sem a necessidade de enviar solicitações constantes ao servidor. Isso é possível graças à combinação de HTML, CSS e JavaScript.

Server-side

(Voltar)

O "server-side" em HTML se refere à parte do processamento que ocorre no servidor antes que o conteúdo seja enviado para o navegador do usuário. No contexto do desenvolvimento web, o server-side envolve a geração dinâmica de páginas web com base em solicitações feitas pelos usuários. Isso é realizado por meio de linguagens de programação do lado do servidor, como PHP, Python, Ruby, Java, entre outras.
